# DhikrFlow

DhikrFlow app with categorized athkar and utilities. Built with Kotlin.

## Requirements
- Android Studio
- JDK 11+

## Run (Debug)
1. Open the project in Android Studio.
2. Let Gradle sync.
3. Run on a device or emulator.

## Secrets
Release signing keys are kept out of git. Keep them outside
this repo and copy them in only when needed.
